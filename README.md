# multistateCodingChallenge

## -Raymond Pantalena

This project provides a function that takes a markdown string and returns a string in html format.

Types of markdown inputs:
\*\*bold\*\*
\_\_underlined\_\_
~~strikethrough~~
\*italic\*
valid email address
valid urls
\*\*\*bold and italic\*\*\*
\*\*bold \_and\_ nested italic\*\*
